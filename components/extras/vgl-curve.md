# VglCurve 

An abstract base component for representing a curve, corresponding
[THREE.Curve](https://threejs.org/docs/index.html#api/extras/core/Curve). 

## props 

- `name` ***name*** (*required*) 

  Name of the component. 

## computed properties 

- `inst` 

  The THREE.Curve instance.
